# Online Tic Tac Toe

A real-time multiplayer Tic Tac Toe game built with React and Node.js + Socket.io.

## How to Play

1. Open the frontend URL
2. Enter your name and a 4-digit room code
3. Share the room code with your friend
4. Start playing!

## Tech Stack

- **Frontend**: React + Vite
- **Backend**: Node.js + Express + Socket.io
- **Hosting**: Vercel (Frontend) + Render (Backend)

## Local Setup

```bash
# Backend
cd backend
npm install
npm run dev

# Frontend (new terminal)
cd frontend
npm install
npm run dev
```
